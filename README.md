# SRGAN for Ocean Salinity Super-Resolution

Ocean salinity is a key parameter in understanding ocean circulation, climate variability, and air-sea interactions. However, satellite-derived salinity products are often limited in spatial resolution.

This work uses:

Satellite SSS dataset 
Auxiliary variables (SST, SSH, Windspeed)
Deep learning (SRGAN)

to reconstruct high-resolution salinity fields from degraded inputs.
